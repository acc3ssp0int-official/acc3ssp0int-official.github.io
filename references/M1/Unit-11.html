Introduction to Secure Web Programming with DJango

  • The reading this week focused on web application development using Django frameworks. Topics covered include an introduction to Django; setting up Django development environment; creating basic website and applications; creating data models and user authentications covered in tutorials on Django of MDN Local Libraries.

This unit focused on introducing me to web application development. It seeked to build on the knowledge gained in the lecturecast by equipping me with practical skills. It explored and implemented some web application components using development frameworks. It also explored technologies and libraries available to implement some of the security solutions discussed during the module.
