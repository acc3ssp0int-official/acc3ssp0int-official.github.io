Unit-3

Approaches to Cyber Security Design

Assignment: Unit-3 Summary Post

This unit focused on approaches to designing Cyber Security solutions. There are various methods and techniques available to develop security solutions. To be able to design effective solutions, it was important to be able to effectively identify the threats and vulnerabilities which would inform the best solution to adopt. We explored these principles that underpin good security designs.

This unit provided the foundation and background knowledge in designing Cyber Security, which was a core part of the module where subsequent units build upon. It serves as the foundation for learning outcomes 2, 3 and 4.
