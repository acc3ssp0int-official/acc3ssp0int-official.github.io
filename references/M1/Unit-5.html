Unit-5

Object-oriented Analysis, Design and Programming Techniques for Secure Systems

  • Discuss the UML concepts and principles to software analysis and design.
  • Explore the concepts, principles and techniques of programming.

This week, the focus was on the implementation of security solutions. The lecturecast introduced object-oriented approaches to secure software design solutions using UML diagrams within the context of Cyber Security. An introduction to programming was also be covered. This included the application of UML object-oriented modelling techniques, such as use case and sequence diagrams, to develop a design document required for project implementation. The introduction to programming covered the principles of programming, as well as the implementation of these principles using the Python programming language.

This unit aimed to introduce me on concepts of how to design software that implements mitigation strategies and security measure to deal with threats and vulnerabilities. It seeked to build on my knowledge to ensure Learning Outcome 3 of the module is achieved.
